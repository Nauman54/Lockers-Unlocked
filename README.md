# 🏀 Lockers Unlocked

A modern basketball memorabilia marketplace and auction platform that connects athletes with fans through secure online auctions and direct product purchases. The platform provides an engaging, responsive, and seamless shopping experience while enabling players to showcase and sell exclusive collectibles.

---

## 📖 Overview

Lockers Unlocked is designed to simplify the buying and selling of basketball collectibles. Players can auction exclusive items, while customers can browse products, place competitive bids, or purchase available items instantly.

The application focuses on delivering an intuitive user experience with fast performance, responsive design, and modern web technologies.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm

---

### Installation

Clone the repository

```bash
git clone https://github.com/Alreadywinner/lockers-unlocked.git
```

Navigate to the project directory

```bash
cd lockers-unlocked
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
